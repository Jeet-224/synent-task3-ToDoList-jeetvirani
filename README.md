# TaskFlow - To-Do Dashboard

A modern and responsive To-Do List Web Application built using **HTML, CSS, and JavaScript**. This project helps users manage daily tasks efficiently with features like task creation, completion tracking, priorities, tags, subtasks, and local storage persistence.

## 🚀 Features

### Task Management

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Clear completed tasks

### Organization

* Priority levels (High, Medium, Low)
* Due dates
* Custom lists
* Tags support
* Subtasks support

### Productivity

* Search tasks
* Filter tasks (All, Active, Completed)
* Sort tasks
* Task counters
* Toast notifications

### User Experience

* Responsive design
* Dark mode support
* Modern dashboard UI
* Mobile-friendly layout
* Smooth animations

### Data Persistence

* Uses Local Storage
* Tasks remain available after page refresh
* Theme preferences are saved automatically

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Lucide Icons
